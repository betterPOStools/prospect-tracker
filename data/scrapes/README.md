# Scrape Archive

Raw JSON exports from Outscraper scrapes. Download from the Queue tab → **Save JSON** button after each completed scrape, then commit here.

Filename format: `scrape-{city}-{state}-{date}.json`

Each file contains:
- Task metadata (city, state, submitted date, query count, record count)
- Full raw results array from Outscraper (pre-import, pre-dedup)
